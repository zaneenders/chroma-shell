# Chroma Shell

Add Chroma Shell to your swift package to give it a try
```swift 
.package(
    url: "https://github.com/zaneenders/chroma-shell.git",
    revision: "main"),

.product(name: "Chroma", package: "chroma-shell"),
```

## Features
[x] Basic ascii color functions
